Creating user, joining team, logging in

What happens invisibly after "Continue":

  1. POST /api/v4/users — create user with {username, email: "{username}@oli.local", password: "OliUser123!"}
  2. POST /api/v4/users/login — login with email + hardcoded password → get auth token
  3. POST /api/v4/teams — create team {name: "{orgName}-team", display_name: "{orgName}", type: "O"}
  4. POST /api/v4/teams/{id}/members — add user to team
  5. Call finishModal({url, name}) → app transitions to main view, already logged in
